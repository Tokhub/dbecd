Dataset name: Individual household electric power consumption Data Set

Data characteristics Multivariate time series Type of building(s) Residential Date 2006-2010
Type of attributes Real numbers Number of buildings 1 Duration 47 months
Level of detail whole building, circuit level Location of buildings France Sampling frequency minutely

Access to the dataset

The data set can be downloaded from this link

Data set description

This data set is one of the different field datasets that are hosted by Centre for machine learning and intelligent systems. It contains 2075259 measurements of electricity consumption data of a household located in France.

Data structure

The data are provided in one csv format file. There exist a 1,25 percent of missing data which are presented by blank spaces in dataset.

Attributes

(Taken from the mentioned link)

Date

Time

Global_active_power: household global minute-averaged active power (kW)

Global_reactive_power: household global minute-averaged reactive power (kW)

Voltage: averaged over a minute (V)

Global_intensity (A)

Sub_metering_1 (Wh) this sub metering includes kitchen appliances

Sub_metering_2 ( Wh) this sub metering includes washing-machine, a tumble-drier, a refrigerator and a light.

sub_metering_3: ( Wh) this sub metering includes an electric water-heater and an air-conditioner.
