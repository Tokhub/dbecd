Access to the dataset

Public access to the raw data is not provided due to privacy concerns. Yet summary of data including various graphs and tables are provided through the reports and tools accessible from:

http://www.branz.co.nz/HEEP

Data set description

This data set is part of the project done by the BRANZ group on New Zealand household energy use measurement and modelling. The focus of the project is given to the way that households consume the energy through the years of study.

The information can be obtained in a straightforward way. First the HEEP Energy and Temperature Data tool should be accessed from the link above. Then user will be required to select graph type and set the x axis and desired filters.
