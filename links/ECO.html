Dataset name: Electricity Consumption & Occupancy (ECO)

Data characteristics Multivariate time series Type of building(s) Residential Date 2012
Type of attributes Real, Integer Number of buildings 6 Duration 8 months
Number of attributes 20 Location of buildings Swiss Sampling frequency 1 Hz for whole power, Once in a few seconds for circuit level

Access to the dataset

The access to the dataset is possible through this link . To get the required credentials for downloading the dataset, person is asked to send contact information and a short description on what they plan to do with the data via e-mail.

Data set description

The ECO data set contains smart meter data (at 1 Hz) from 6 households in Switzerland collected over a period of 8 months. It further contains plug data of 6 to 10 plugs per household (also measured at 1 Hz) and information about the occupancy of people in the house (indicated through a tablet PC that was located next to the entrance door). It is currently being used by >40 researchers

Dataset structure

The UMass smart* data set includes several compressed .tar files for each home. And each one of them contains several csv-Excel files for every day and a Format file describing the attributes name.

Wind Speed, wind Direction, wind Gust and wind Gust Direction
