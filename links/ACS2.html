Data set description

This dataset is composed of the power consumption of 225 different models/brands electronic appliances and no whole consumption (aggregated) data. The appliances are classified to 15 categories of fridges and freezers, TVs,Hi-Fi systems, Laptops, Computer stations (including monitors), compact fluorescent lamps (CFL), microwaves, coffee machines, mobile phones, printers,incandescent lamps, shavers, fans, kettles and monitors.

Dataset structure

The ACS2-F2 data are presented in two groups of file formats which are XML and MAT. The XML file contains raw data and ground truth data while the MAT format contains raw data easily usable in MATLAB
