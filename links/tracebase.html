Access to the dataset

The dataset is freely available from this link . Though, a registration process that includes providing name ,email and affiliation is required. Then the download link and a password will be sent to the provided email.

Data set description

This dataset is composed of the power consumption of 158 smart metered electrical appliances and no whole consumption (aggregated) data. This set contains 43 different types of appliances.

Dataset structure

The data are grouped in three subsets of complete, incomplete and synthetic .zip files. Details of each one is given in a Readme.txt existing in the directory. Each group of data contains folders named by the appliances type, like printer, dishwasher, etc. then every folder contains consumption data in csv format.
